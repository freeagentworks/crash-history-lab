# Project Charter

## Project Name
Crash History Lab

## Mission
投資判断を直接出すのではなく、暴落の構造を可視化・比較し、研究と教育の質を上げる。

## Target Users
- 初期: 限定公開ユーザー
- 将来: 研究コミュニティ向け拡張を想定

## Non-Goals (MVP)
- リアルタイム取引執行
- 個別ユーザー認証/権限管理
- 分足・高頻度データ対応

## Success Metrics
- 主要イベント（例: 2008, 2020, 2022 など）を再現表示できる
- 暴落判定結果がユーザー感覚と大きく乖離しない
- 類似局面検索の上位候補が学習用途で妥当
- 主要画面操作を3クリック以内で完了できる

## Deployment Policy
- 初期公開: Vercel URLを知る人向け（認証なし）
- 免責文を全画面に常時表示
