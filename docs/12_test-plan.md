# Test Plan

## Unit Tests
- 指標計算の数式検証
- 欠損値・ウォームアップ期間の挙動
- スコア正規化と重み再正規化
- 類似度計算（特徴量距離/DTW）
- バックテスト指標計算

## Integration Tests
- Yahoo取得 -> 指標 -> イベント抽出の一連フロー
- 比較画面4件同期表示
- 類似検索APIの応答形式

## UI Tests
- 主要画面の表示崩れ
- パラメータ変更時の再描画
- エラーハンドリング表示

## Data Regression Tests
- 既知イベント（例: 2020-03周辺）でランキング上位を維持
- 閾値変更時のイベント件数変化が単調性を満たす
