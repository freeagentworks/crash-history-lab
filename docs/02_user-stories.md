# User Stories

## Persona A: 研究者
- 日経平均の過去暴落を一覧化し、イベントごとの前後挙動を比較したい
- 同一銘柄内の類似局面を見つけ、構造的再現性を検証したい

## Persona B: 教育コンテンツ作成者
- 受講者に見せる教材として、暴落の典型パターンを選びたい
- 同じ指標群で市場間（株・指数・暗号・FX）を横断比較したい

## User Stories
1. ユーザーとして、ティッカーを入力し、暴落ランキングを確認したい。
2. ユーザーとして、イベント日を選び、前後チャートを見たい。
3. ユーザーとして、同時に4イベントを並べて比較したい。
4. ユーザーとして、似ている過去局面を自動提案してほしい。
5. ユーザーとして、指標パラメータを調整して判定感度を変えたい。
6. ユーザーとして、テンプレート戦略の簡易検証結果を見たい。
7. ユーザーとして、データや機能が投資助言ではないことを明確に認識したい。
