# Crash History Lab

暴落局面を「前後の文脈」で学習する研究・教育向けWebアプリ（Next.js 16 + TypeScript）。

## Purpose
- 暴落イベントを定量検出する
- イベント前後のチャート挙動を比較学習する
- 類似局面を探索して再現性を確認する

## Product Scope (MVP)
- 日足データ（Yahoo Finance）
- 自由入力ティッカー + プリセット選択
- 暴落判定（単一条件 / スコア）
- イベント比較（同時4件）
- 類似局面検索（同一銘柄内）
- 簡易バックテスト（テンプレート方式）
